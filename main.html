Machine Learning

Moving towards the future- that is the focus of this chapter. We have gone through a brief history of bioinformatics and dabbled upon current topics. Now, we will set our thoughts upon the future. One area of current research is in machine learning. Although the definition of such can vary between individuals in different fields, we can think of machine learning as a way for programs to improve its performance without prior, static instructions from a human programmer. In essence, it is dynamically self-programming without the use of loops, which are pre-defined by the programmer. Ideally, machine learning can be performed through tasks or with “human” supervision. Human supervision can be through the use of training sets, finding a rule to label, using a semi-relative training set, or be pre-disposed to reinforcing outcomes. Tasks can include classifying, regressing, clustering, or performing dimensionality reduction. The clustering results from scRNA-seq can be visualized using PCA, t-SNE, and UMAP.  

t-SNE

Dimensionality reduction techniques include t-distributed stochastic neighbor embedding (t-SNE), which was first pioneered in 2002 and improved upon by introducing symmetric cost function with student t-distribution in calculating similarity. Essentially, it is an algorithm in high dimension (Euclidean distance) that calculates conditional probabilities, which in turn calculates similarity and minimizes the cost function by gradient descent method. This technique can be performed using R with the Rstne and ggplot2 libraries. The Rstne library will perform calculations, while the ggplot2 library visualizes the results. For more information on t-SNE, watch the video below.   

UMAP

The other technique to mention is Uniform Manifold Approximation and Projection (UMAP). UMAP can be used in R with the umap library. First, begin by constructing a fuzzy topological representation of real data. Next, optimize a low dimensional representation to the fuzzy topological representation. Lastly, use PCA to visualize the clustering of data, which can also be performed in R. For more information on UMAP, watch the video below.
